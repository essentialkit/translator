Xtension Localizer

Automatically translate browser extensions

17.9% of internet users are fluent in English. For context, English is the most popular language in the world followed by Chinese at 14.3%. This highlights the importance of i18n to reaching a global audience.

9% of chrome extensions are available in more than one language, based on data from chrome-stats.com. This underscores the difficulty of internationalizing a browser extension

This tool works for all addons that follow the WebExtensions spec, including Chrome extensions, Firefox addons, Edge addons and Opera addons.

For information on how to internationalize your extension, see the Chrome i18n documentation.

1. Upload Extension

The zip file should include a manifest.json file in its root directory. If the manifest.json file specifies a default_locale, it would be used for translation.

OR

2. Translate

Translate extension to any of the 52 supported languages on the Chrome Webstore by selecting target locales.

Deselect All

3. Download

Click download to save your translated extension.
